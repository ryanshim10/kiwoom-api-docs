## 🚀 주요 API 흐름 (매매)

### 1단계: 인증
```python
# OAuth 토큰 발급
POST /oauth2/token
```

### 2단계: 시세 조회
```python
# 현재가 조회
GET /api/dostk/inquire/price

# 차트 데이터
GET /api/dostk/inquire/chart
```

### 3단계: 주문
```python
# 주문 (kt10001: 매도, kt10000: 매수)
POST /api/dostk/ordr
```

### 4단계: 정정/취소
```python
# 정정 (kt10002)
POST /api/dostk/ordr

# 취소 (kt10003)
POST /api/dostk/ordr
```

### 5단계: 체결 확인
```python
# 미체결/체결 조회 (ka10075, ka10076)
GET /api/dostk/inquire/order
```

## 📌 주의사항

### Rate Limit
- 초당: 최대 10 req/sec
- 분당: 최대 600 req/min

### 인증
- 모든 요청에 Authorization Header 필수
- Format: `Bearer {ACCESS_TOKEN}`

### 모의투자
- `https://mockapi.kiwoom.com` 사용 (KRX만 지원)
- 실전: `https://api.kiwoom.com`
